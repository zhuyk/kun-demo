# 香港虚拟银行线上开户模拟指南

一个移动端友好的 H5 网页，帮助内地居民在去香港前熟悉**虚拟银行线上开户**的全流程。

## 覆盖银行

- **众安银行 ZA Bank** - 香港最大虚拟银行，最快2分钟开户
- **天星银行 Airstar Bank** - 小米+富途合资，支持港美股联动

## 模拟流程

1. 行前准备（材料清单）
2. 选择银行 & 下载App
3. 填写个人信息（模拟表单）
4. 证件拍照 & 人脸识别
5. 开户成功 & 后续注意事项

## 使用方式

直接用手机浏览器打开即可：

**在线访问：** https://zhuyk.github.io/kun-demo/

或本地打开 `index.html`。

## 技术栈

纯 HTML + CSS + JavaScript，无需任何构建工具或依赖。
